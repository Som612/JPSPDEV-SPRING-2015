====README============

1) To run the project, please unzip both the parts - Project1Unit6 Server Database and Project 1 Unit 6 Servlet into eclipse

2) The local SQL connection should be in service with the username = root. The password for the database is 'Java@18641'.

3) The project is executed through the driver as provided in the driver package.

4) Output can be seen for creating, updating and deleting the database for both .txt files and .Properties files.

5) The database is called save_automobile. It is dropped everytime after all the operations are done to ensure the driver can be run multiple times. 

6) The DatabaseIO method in FileIO package has the implementation of the database access methods.

7) The JDBC connector jar is in the lib folder and the buildpath is pre-configured.
